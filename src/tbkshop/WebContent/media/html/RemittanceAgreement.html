产品合作授权协议

授权方：开发者 (以下简称为甲方)

被授权方：广州市谷果软件技术有限公司（以下简称为乙方）

一、合作方式

1. 甲方将其研发并拥有的完全知识产权的基于Android版本的客户端产品(以下简称“合作产品”)植入乙方SDK代码，即认定为授予乙方代理。

2. 经甲方授权，乙方可以免费发布在中国移动、中国电信、中国联通等国内外的软件商城。

3. 合作期间，由甲方合作产品产生的所有广告收入，乙方全额支付给甲方，支付金额及结算数据以“酷果平台”的数据为准, 合作产品中出现的广告单价以“酷果平台”公布的单价为结算依据。

4. 甲方定期根据“酷果平台”统计的数据申请结算，每周周三为酷果结算日,凡每周三中午12点之前申请汇款经乙方确认后2个工作日内支付到甲方在“酷果平台”登记的指定账户,如遇节假日则正常顺延至下一个结算日结算。如果收款方为个人账户，乙方汇款时将代扣0.5%的汇款手续费和5.5%的营业税。如果收款方为对公账户，需要在汇款之前开具相应金额的发票，免扣税及手续费。

 

二、权利与义务

（一）、甲方权利与义务

1. 甲方保证合作产品内容符合中国的法律规定，不包括任何色情、政治等非法信息，不存在盗取、破坏用户数据及系统的隐藏内容。甲方保证其具备合法资格从事本合同规定的服务，合作产品具有合法版权以及其他权利，不违反任何法律法规，也不侵犯任何第三方的合法权益。甲方不得使用乙方之服务销售非法商品，包括但不限于通过黑客、淫秽网站等形式进行非法商业行为或发布应用软件等。因违反上述内容而造成的一切责任及后果，均由甲方自行承担，如因此造成乙方损失的，甲方应当承担全部赔偿责任，包括乙方的直接损失、预期可得利益损失以及乙方为此支出的诉讼费、律师费等相关费用。

2. 甲方负责合作产品开发和测试的全部过程，并植入乙方提供的SDK代码，同时承诺所有合作产品中的所有语言种类都支持中文。甲方应提供合作产品的全面的支持文件，有权对合作产品进行更新、完善、升级和修改，并确保合作产品的稳定运行。乙方可要求甲方对不符合要求的产品进行无偿修正，甲方应给予积极配合进行产品改进。

3. 甲方严格遵守《全国人民代表大会常务委员会关于维护互联网安全的决定》、《中华人民共和国电信条例》、《互联网信息服务管理办法》等法律法规的有关规定，依法从事互联网及移动互联网服务。

4. 甲方严格遵守国家《保密法》等有关法律法规，不泄漏国家秘密，不泄漏用户个人资料。

5. 甲方严格遵守酷果平台合作产品的相关规定，承诺合作产品中不出现淫秽、色情、六合彩等任何违规行为,其中淫秽、色情、低俗等违规内容的定义包括但不限于以下内容：表现或隐晦表现性行为、令人产生性联想、具有挑逗性或者污辱性的内容；对人体性部位的直接暴露和描写；对性行为、性过程、性方式的描述或者带有性暗示、性挑逗的语言；对性部位描述、暴露，或者只用很小遮盖物的内容；全身或者隐私部位未着衣物，仅用肢体遮盖隐私部位的内容；带有侵犯个人隐私性质的走光、偷拍、漏点等内容；相关部门禁止传播的色情、低俗小说，音xx内容，包括一些电影的删节片断；情色小说、图片、xx、语音、动漫，以及一夜情、换妻、SM等不正当交友信息。

6. 甲方保证所有在酷果平台登记的信息真实、合法。甲方不得采用任何不正当手段损害酷果平台的利益及声誉。

二）、乙方权利与义务

1. 为了服务的顺利、平稳进行，乙方应尽最大努力保证服务平台的合法性和平台运行的稳定性。

2. 乙方负责提供甲方的合作产品所需广告；乙方负责保障提供给甲方的各项后台数据的准确性与真实性，并负责按期向甲方结算广告费用。

3. 根据市场运营情况，乙方有权要求甲方对合作产品依市场运营实际情况进行适当更改或升级，或提供不同语言的版本，具体情况及成本负担由双方另立补充协议进行详细说明。

4. 乙方保证其具备合法资格从事本合同规定的服务，向甲方合作产品提供的相关推广和运营合法，不违反任何法律法规，也不侵犯任何第三方的合法权益。

三、知识产权

1. 甲乙双方彼此应对本次合作内容中所涉及的对方知识产权予以充分尊重和保护，并保证在履行本协议时，对于所涉及的第三方知识产权予以了充分的尊重和保护，并各自具有完整的知识产权。

2. 甲方授权给乙方的所有合作产品（包括但不限于许可软件中所含任何声音、音乐、图像、照片、动画、录像、xx软件以及应用程序），所有权仍归甲方所有，包括但不限于专利、著作权等知识产权，并不因双方的合作而有所改变。

3. 因双方合作需要，由乙方独立开发的技术、开发工具、宣传资料等权益，归乙方所有，并不因双方的合作而有所改变。

4. 甲方对本协议项下合作产品享有完整的知识产权，乙方不应以任何方式从事危害甲方知识产权的活动。

5. 协议双方仅可在本协议约定的范围内，使用对方的商标、LOGO等，不得以任何其他目的或方式使用对方的商标、logo等。

四、保密

1. 未经对方书面许可，任何一方不得向第三方泄露本协议的条款的任何内容以及本协议的签订及履行情况，以及通过签订和履行本协议而获知的对方及对方关联公司的任何信息。

2. 保密信息不包括：（Ａ）已被公众掌握的信息或材料；（Ｂ）双方通过合法途径依法掌握的信息或材料，且当时双方并不负有任何保密义务。

3. 上述保密条款不因本协议的无效、提前终止、解除或不具操作性而失效

五、违约责任

1. 双方应严格按照法律规定及本协议约定履行本协议义务。当事人一方不履行本协议义务或者履行本协议义务不符合约定的，守约方有权以书面方式通知并要求违约方在限期内有效纠正违约行为，并承担继续履行、采取补救措施或者赔偿损失等违约责任。违约方在收到守约方具体说明违约情况的书面通知书后，如确认违约行为实际存在，则应在3日内对违约行为予以纠正并书面通知对方；如认为违约行为不存在，则应在3日内向对方提出书面异议或说明。在此情形下，甲乙双方可就此问题进行协商。如违约方在上述限期内未能有效纠正或说明其违约行为的，守约方有权解除本协议，并要求违约方就违约事项赔偿损失。

2. 因用户使用合作产品而导致任何第三方向任何一方提起索赔要求、诉讼或其他侵权指控行为，政府机构对任何一方做出处理/处罚，双方应互相配合进行处理。因违反合同、明显技术缺陷等过错等造成发生问题的一方应承担由此造成的一切责任、损失和赔偿，与另一方无关，包括但不限于诉讼费用、律师费用、差旅费用、和解金额、罚款或生效法律文书中规定的损害赔偿金额、软件使用费等全部损失等。一方给另一方造成名誉损失的，应当赔偿另一方名誉损失费用。

六、不可抗力因素

1. 不可抗力是指合同双方不可预见、不能避免并不能克服的客观情况，该事件妨碍、影响或延误任何一方根据合同履行其全部或部分义务。该事件包括但不限于政府行为、自然灾害、战争、网络堵塞或中断、黑客袭击或任何其他类似事件。

2. 由于以上所述不可抗力事件致使合同的部分或全部不能履行或延迟履行，则双方于彼此间不承担任何违约责任。当事人迟延履行后发生不可抗力的，不能免除责任。

七、协议生效方式

在甲方勾选“我同意”的选项之后，双方就各自享有公平的权利，并承担各自的义务。

八、有效期

所有上传至酷果平台的产品，在产品存续期，均默认为在授权期内。

九、有效区域

有效区域为中国地区，但乙方不保证乙方所有用户均来自有效区域以内。

十、其他

凡因执行本协议所产生之争议或与之有关的其他争议，双方应平等、友好协商，以合理解决问题的态度积极解决问题。如双方不能协商解决，可就该争议提交广州仲裁委员会仲裁。
